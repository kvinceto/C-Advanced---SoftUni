2.	Average Student Grades
Create a program, which reads a name of a student and his/her grades and adds them to the student record, 
then prints the students' names with their grades and their average grade.
Examples
Input				Output
7
John 5.20
Maria 5.50
John 3.20
Maria 2.50
Sam 2.00
Maria 3.46
Sam 3.00	
			John -> 5.20 3.20 (avg: 4.20)
			Maria -> 5.50 2.50 3.46 (avg: 3.82)
			Sam -> 2.00 3.00 (avg: 2.50)

4
Vlad 4.50
Peter 3.00
Vlad 5.00
Peter 3.66	
			Vlad -> 4.50 5.00 (avg: 4.75)
			Peter -> 3.00 3.66 (avg: 3.33)
5
George 6.00
George 5.50
George 6.00
John 4.40
Peter 3.30	
			George -> 6.00 5.50 6.00 (avg: 5.83)
			John -> 4.40 (avg: 4.40)
			Peter -> 3.30 (avg: 3.30)
