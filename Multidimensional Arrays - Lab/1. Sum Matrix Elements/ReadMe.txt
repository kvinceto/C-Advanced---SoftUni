1.	Sum Matrix Elements
Write a program that reads a matrix from the console and prints:
•	Count of rows
•	Count of columns
•	Sum of all matrix elements
On the first line, you will get matrix sizes in format [rows, columns]
Examples
Input			Output
3, 6
7, 1, 3, 3, 2, 1
1, 3, 9, 8, 5, 6
4, 6, 7, 9, 1, 0
			3
			6
			76
