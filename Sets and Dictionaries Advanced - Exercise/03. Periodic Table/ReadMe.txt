Problem 3.	Periodic Table
Create a program that keeps all the unique chemical elements. On the first line, 
you will be given a number n - the count of input lines that you are going to receive. 
On the next n lines, you will be receiving chemical compounds, separated by a single space. 
Your task is to print all the unique ones in ascending order:
Examples
Input	Output
4
Ce O
Mo O Ce
Ee
Mo
	Ce Ee Mo O
