1.	Count Same Values in Array
Create a program that counts in a given array of double values the number of occurrences of each value. 
Examples
Input					Output
-2.5 4 3 -2.5 -5.5 4 3 3 -2.5 3
					-2.5 - 3 times
					4 - 2 times
					3 - 4 times
					-5.5 - 1 times
2 4 4 5 5 2 3 3 4 4 3 3 4 3 5 3 2 5 4 3
	
					2 - 3 times
					4 - 6 times
					5 - 4 times
					3 - 7 times
